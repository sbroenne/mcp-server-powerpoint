Add parameterized shape effects: drop shadow (color/transparency/blur/offset), glow, reflection, soft edge, and 3D bevel. Extends the existing `set-shadow`/`get-shadow` actions with optional formatting parameters (backward compatible) and adds new `set-glow`/`get-glow`, `set-reflection`/`get-reflection`, `set-soft-edge`/`get-soft-edge`, and `set-bevel`/`get-bevel` actions to the `shape` MCP tool.
